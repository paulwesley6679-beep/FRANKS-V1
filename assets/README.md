# Vendor Marketplace

A lightweight marketplace platform that connects vendors and buyer through direct whatsApp transaction.

This is VERSION 1 (MVP) built  using pure HTML,CSS and Javascript.

## Vision 

To create a simple and accesible online marketplace where:

- Vendors can upload products easily
- Buyers can browse and filter products
- Transaction happen directly via WhatsApp
- No complex payment systems
- No friction

This verison focuses on simplicity, speed and validation.

## MVP features 

### Landing page
- Clear value proposition
- Simple call-to-action
- Role selection (vendors / Buyers)

### Marketplace
- Category filtering
- Subcategory filtering
- Trending product shuffle
- Direct whatsApp purchase button
- Dynamic product rendering 

### Seller Dashboard
- Upload product form
- Dynamic subcategory selection
- Instant product preview
- Product auto-added to marketplace
- Clean UI structure

### Feedback Page
- collect user feedback
- Role-based feedback (vendor / Buyer)
- Console logging for MVP validation

## Tech Stack
- HTML5
- CSS3 (custom layout system)
- Vanilla Javascript (ES6)
- No frameworks
- Np backend (frontend-only MVP)
